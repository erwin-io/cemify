How to Use

Getting Started

To get started, follow these steps:

  1. Request a user account from the admin.
  2. Wait for the admin to create your account.
  3. Once your account is ready, log in with the provided temporary credentials.

How to Reset Password

If you need to reset your password, follow these steps:

  1. Contact the admin or support team to request a password reset.
  2. Provide necessary information for identity verification.
  3. Wait for the admin or support team to handle your request.
  4. Follow the instructions provided to reset your password.

FAQs

Commonly asked questions about our platform:


How do I request a user account?

To request a user account, please contact our admin team via email or phone. They will assist you in creating an account and provide you with the necessary credentials to log in.

How long does it take for my account to be ready after requesting?

The time it takes for your account to be ready may vary depending on administrative processes. Our team strives to process requests promptly, and you will receive an email notification once your account is created.

Can I access all features immediately after logging in?

No, upon logging in, you can only access the how to use page. Some advanced features may require additional permissions or administrative approval. If you encounter any restrictions, please contact the admin for assistance.

How can I generate reports for burial records?

To generate reports for burial records, navigate to the "Burial Records" section, click the generate report button and {{title}} will generate a comprehensive report for you to review.

How do I generate a burial record certificate?

Generating a burial record certificate is simple. Navigate to the specific burial record for which you need a certificate, then locate the option to generate a certificate within the burial details page. Follow the prompts to customize the certificate, and our system will generate it for you instantly.

How do I navigate to specific features such as cemetery mapping or burial records?

{{title}}'s user-friendly interface allows you to navigate easily to specific features. Simply use the main menu to access different sections such as cemetery mapping, burial records, reservation, work orders, and user management.

What if I forget my password?

If you forget your password, please contact the admin or support team for assistance.

How can I learn more about each feature and its usage?

Each feature on our platform comes with detailed instructions on how to use it effectively. You can find this information in the How to use this feature section page of how to page, which provides step-by-step guides for each feature.

Cemetery Mapping

Visualize and manage cemetery lot information on an interactive map.


How to use Cemetery Mapping

To use the cemetery mapping, follow these steps:

  1. Navigate to the "Cemetery Mapping" section.
  2. Explore the map by zooming in/out and dragging.
  3. Click on a lot to view detailed information.
  4. Use the search function to search burial records.

Burial Records

Manage and maintain burial records, deactivate or removing records as necessary. Generate burial record certificate or reports.


How to use Burial Records

To use the burial records, follow these steps:

  1. Access the "Burial Records" section from the main menu.
  2. Add burial record by entering burial details.
  3. Update existing record as needed.
  4. Track burial recrods and generate certificate or reports.

Manage Lot Reservation

Manage incoming lot reservation requests from clients with approval and reject functionality.


How to manage Lot Reservation

To manage Lot Reservation, follow these steps:

  1. Go to the "Reservation" feature from the main menu.
  2. View the list of incoming reservation requests.
  3. Review each request and approve or reject it accordingly.
  4. For approved reservations, allocate the lot to the client and confirm the reservation.

Work Order

Create and manage work orders for staff members.


How to use Work Order

To use the Work Order, follow these steps:

  1. Access the "Work Order" section from the main menu.
  2. Create a new work order by providing details of the task.
  3. Assign the task to a specific staff member.
  4. Track the progress of the work order and mark it as complete when done.

User and Access Management

Administer user accounts, permissions, and roles.


How to manage Users and Access

To manage the Users and Access, follow these steps:

  1. Go to the "User Management" panel accessible to admin users.
  2. Add new users by entering their details.
  3. Assign access and permissions to users based on their responsibilities.
  4. Manage user accounts and password.
